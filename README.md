## 📌 Project Overview

This project is a Netflix Data Analysis Dashboard built using Microsoft Excel. The dataset was cleaned and analyzed to generate meaningful insights using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

The dashboard provides an interactive view of Netflix content based on genres, countries, ratings, release years, and content types.

---

## 📊 Features

- Total Titles KPI
- Total Movies & TV Shows
- Average Rating
- Genre-wise Content Analysis
- Country-wise Content Distribution
- Release Year Trend Analysis
- Average Duration Analysis
- Interactive Slicers for filtering data
